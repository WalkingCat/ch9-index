(Live Unit Testing) [0:05:55] [2017/03/13]
Visual Studio 2017 的 Live Unit Testing (即時單元測試) 功能，大大的降低撰寫單元測試程式潛在的開發成本，當即時單元測試功能啟動後，所有的單元測試程式都會自動在背景不斷執行，即便程式碼尚未存檔，單元測試還是會自動驗證並即時回饋程式碼的測試執行結果，讓我們可以更流暢的實現 TDD 測試先行開發，提升開發效率與程式品質。


[10LiveArchitectureDependencyValidation] [0:06:40] [2017/03/13]
Visual Studio 2017 的即時相依性檢測功能，可以幫助架構師在規劃專案架構時用直覺的圖表進行設計，並搭配 Visual Studio 2017 內建的即時程式碼分析功能 (Live Code Analysis)，在背景即時分析專案中每一段程式碼是否符合專案的架構設計，也能幫助開發團隊成員盡早得知他們所寫的程式碼已經發生了架構違規狀況，大大的降低團隊開發的生產力與軟體品質。
