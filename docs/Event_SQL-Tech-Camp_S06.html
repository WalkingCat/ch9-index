R, ? [1:04:38] [2016/08/26]
뜨고 있다는 R, 배우려고 한 번 했으나 뭐가 이리 어렵고 복잡한지, 놀라서 멈칫하셨던 분들 위해 R이 무엇인지 알아보고자 합니다. R의 특징, R의 유용성을 코드를 돌려가며 번갯불에 R 볶아 맛 한번 보겠습니다. 통계 모르는 R 왕초보에게 특화되어 있습니다. 이제 SQL Server에 탑재된 R도 활용해 보실 수 있는 기회입니다.
