Series - Migracja-usug-do-Windows-Server-2012-R2


Migracja roli WSUS [2015/03/20]
W module 6 dowiesz się jak zmodyfikować obecną strukturę serwerów WSUS, która zapewni możliwość migracji do nowszej wersji systemu operacyjnego.


Migracja roli WDS Server [2015/03/20]
W module 5 zobaczysz w jaki sposób zmigrować obrazy wdrażania systemu Windows oraz obrazy startowe do nowszej wersji systemu Windows Server.


Migracja Distributed File System [2015/03/20]
W module 4 zostanie zaprezentowany sposób migracji usługi DFS, polegający na rozbudowie obecnej konfiguracji o nową wersję sytemu serwerowego.


Migracja Print and Document Services [2015/03/20]
W module 3 przedstawiony zostanie sposób przeniesienia centralnego serwera wydruków. Pomiędzy wersjami Windows Server 2008 R2 a Windows Server 2012 R2.


Migracja Active Directory [2015/03/20]
W module 2 zaprezentowany zostanie sposób migracji usługi katalogowej Acyive Directory. Wskazane zostaną główne aspekty, które należy wziąć pod uwagę podczas migracji tej usługi.


Migracja roli DHCP Server [2015/03/20]
W module 1 dowiesz się z czego składa się i jak działa usługą DHCP Server. Zobaczysz proponowany sposób migracji usługi pomiędzy różnymi wersjami systemów operacyjnych.
